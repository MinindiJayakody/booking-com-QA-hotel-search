# Booking.com Hotel Search - QA Project

**Manual Testing & Test Management Project**  
Third-Year IT Undergraduate | BCI Campus

### 📋 Project Overview
Comprehensive quality assurance project for Booking.com hotel search functionality.  
Tested multiple Sri Lankan destinations (Kandy, Polonnaruwa, Galle, Nuwara Eliya) with various dates, group sizes (up to 6 adults + 5 children), and room configurations (1–6 rooms).

### 🛠️ Tools & Technologies
- TestLink (Test Case Management)
- Mind Mapping (XMind / MindMeister)
- Manual Testing
